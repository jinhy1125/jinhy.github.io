桌面工具

Fences

Fences即“栅栏”，它可以轻松的让我们在桌面创建一个个小方块然后我们可以把我们桌面的东西归类到里面。它还提供了自动归类的功能，但是当然不能满足我们的需求。我们还是要在根据自己的需求进行调整的。

Fences还有一个功能是直接拉一个文件夹过来做一个框，相当于你在桌面上固定了一个文件管理。图中的PT其实就是链到桌面的一个文件夹。

如果双击空白区域的话会隐藏桌面上的所有东西，还你一片净土。

下面是我的页面~

Fences官网

SAO Utils

SAO Utils是一个仿刀剑神域番剧里面游戏菜单的一个启动器工具。集成了各种酷炫的功能，开机还会有开机动画跟桐人进入潜行世界的效果一样，如果你中二的话你肯定喜欢这个开机动画，不喜欢建议关掉，太浮夸了。我自己感觉的话，如果这个软件用到极致的话，基本上就不需要用Fences了，桌面上只要放置一些临时文件就ok。

上图桌面上的那些生命条，内存监控什么的就是由这个软件提供的。还有番剧提醒，立绘，音乐播放提醒，邮件收发等等各种插件。如果觉得还不满足，那可以自己开发呀~总而言之这个软件特别的6.

SAO Utils官网

Everything官网

Wox

一个极简的启动器，它的样子你可能在Mac OS上见过。配合Everything使用效果极佳。
也有很多便捷的功能和插件，像计算器啊、有道翻译啊、剪切板啊、二维码啊、github啊等等等等。。。

还可以置顶一些你搜索到的项目。

Wox官网

Clover

可以给windows的资源管理器加书签栏。以后找什么东西终于不用东翻西找啦。虽然我觉得对我来说没有起到太大的效果。不过像我这样的估计也不多见。不止可以挂本地的文件夹，也可以挂ftp和远程目录。

Clover官网
